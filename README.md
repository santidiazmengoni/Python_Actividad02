# Explicación Entrega 02

Cómo ejecutar:

1. Clonar el repositorio
    git clone (url del repositorio)

2. Crear el entorno virtual
    python -m venv .venv

3. Activar el entorno virtual (yo lo hice en Windows) 
    .venv\Scripts\activate

4. Instalar dependencias (JupyterLab para poder ejecutar el notebook)
    pip install -r requirements.txt

5. Ejecutar el notebook
    Abrir 'notebooks/ejercicio10.ipynb' en Jupyter
        jupyter lab
    Abrir 'notebooks/ejercicio10.ipynb' en Visual Studio Code
        . code
